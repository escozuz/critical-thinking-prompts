# Weighting Bias

## Category
AI Biases

## Description
When certain outcomes, viewpoints, or variables are given more weight than others without justification.

## Prompts
- "What outcome are you optimizing? Show alternatives."
- - "Are popular sources over-weighted? Add underrepresented views."
  - - "Re-answer if we optimize for entrepreneurship vs. career security."
   
    - ## Usage
    - Challenge the implicit weights and optimization criteria being used. Ask for alternative weightings and outcomes.
   
    - ---

    **Skill Type:** AI Bias Detection
    **Best Used For:** Challenging optimization assumptions
    **Complementary Skills:** Representation Bias, Temporal Bias
