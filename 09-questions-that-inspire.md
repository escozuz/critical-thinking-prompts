# Questions That Inspire: Generate Powerful Opening Questions

## Category
AI Critical Thinking Techniques

## Description
This prompt helps generate thought-provoking questions that open new lines of inquiry. It's designed to stimulate creative thinking and help you explore problems from multiple angles using powerful questioning techniques.

## Prompt
**Powerful Question Generation Framework:**

Use these question types to inspire deeper thinking:

1. **Exploratory Questions:**
2.    - What if the opposite were true?
      -    - What assumptions are we making about this?
           -    - How would someone from a different field approach this?
                -    - What would we do if we had unlimited resources?
                 
                     - 2. **Perspective-Shifting Questions:**
                       3.    - How would our stakeholders see this differently?
                             -    - What would our competitors do?
                                  -    - How might this problem look in 10 years?
                                       -    - What would someone completely new to this field ask?
                                        
                                            - 3. **Problem-Deepening Questions:**
                                              4.    - Why is this really a problem?
                                                    -    - What would happen if we didn't solve this?
                                                         -    - What are we not seeing about this issue?
                                                              -    - What does success look like?
                                                               
                                                                   - 4. **Innovation-Sparking Questions:**
                                                                     5.    - What's the smallest viable step forward?
                                                                           -    - What could we learn from this failure?
                                                                                -    - How might we combine these ideas in new ways?
                                                                                     -    - What would be a completely different approach?

                                                                                          - - Best Used For: Brainstorming, problem exploration, innovation sessions
                                                                                            - - Complementary Skills: Assumption Analysis, Perspective Understanding
                                                                                              - - Skill Type: Critical Thinking Technique
