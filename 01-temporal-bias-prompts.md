# Temporal Bias Prompts

## Category
AI Biases

## Description
When conclusions rely on outdated information or ignore recent changes in context.

## Prompts

### Prompt 1
"What's your latest data cutoff for this answer?"

### Prompt 2  
"List 2024–2025 sources; flag older ones."

### Prompt 3
"What recent shifts could reverse this recommendation?"

## Usage
Use these prompts when analyzing information that could be affected by recent developments or changes in data sources. Ask for confirmation of the recency of information used and request that the analysis incorporates the latest available data.

---

**Skill Type:** AI Bias Detection  
**Best Used For:** Ensuring current, up-to-date analysis  
**Complementary Skills:** Representation Bias, Weighting Bias
