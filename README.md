# Critical Thinking Prompts & Skills Library

A comprehensive collection of 15 critical thinking prompts organized into three categories: **AI Biases**, **Human Biases**, and **AI Critical Thinking** techniques.

## 📁 Folder Structure

### Bias Awareness Prompts (6 files)

**AI Biases:**
- `01-temporal-bias-prompts.md` - Address outdated information and recency bias
- `02-representation-bias-prompts.md` - Ensure diverse perspectives are included
- `03-weighting-bias.md` - Challenge outcome optimization assumptions

**Human Biases:**
- `04-confirmation-bias.md` - Confront opposing viewpoints directly
- `05-availability-bias.md` - Incorporate long-term and historical context
- `06-authority-bias.md` - Evaluate claims independently from their source

### Critical Thinking Techniques (7 files)

- `07-set-boundaries.md` - B: Define AI vs. human task boundaries
- `08-assessing-information.md` - I: Evaluate reliability, relevance, and validity
- `09-questions-that-inspire.md` - Q: Generate powerful opening questions
- `10-understand-perspectives.md` - P: Simulate diverse viewpoints
- `11-identify-assumptions.md` - A: Surface hidden assumptions
- `12-clarify-concepts.md` - C: Establish shared understanding
- `13-root-cause-analysis.md` - C: Understand problems at the root level

### System Prompts (2 files)

- `14-critical-thinking-mode.md` - System prompt for critical thinking assistant
- `15-personal-consultant-coach.md` - Strategic thought partner framework

## 🚀 How to Use These Files

1. Each file is standalone and can be used independently
2. Share specific prompts with your team for critical thinking exercises
3. Use system prompts to set up AI assistants with critical thinking capabilities
4. Combine prompts for comprehensive problem-solving workflows

## 📖 File Format

Each skill file includes:
- **Title** - Name of the prompt/skill
- **Category** - Classification (AI Biases, Human Biases, AI Critical Thinking, System Prompt, or Personal Coach)
- **Description** - Brief explanation of what it addresses
- **Detailed Content** - Full prompts, instructions, and guidance
